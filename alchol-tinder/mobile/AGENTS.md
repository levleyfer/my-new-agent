# Expo HAS CHANGED

Read the exact versioned docs at https://docs.expo.dev/versions/v54.0.0/ before writing any code.

The project was downgraded from SDK 56 to SDK 54 on 2026-06-21 because the App Store
build of Expo Go did not yet support SDK 56 (Apple review lag) and a separate Expo Go
56.0.1 Android bug rejected all SDK 56 projects outright. Do not upgrade back to 56
without first checking whether Expo Go now supports it.
